Update Script
-------------

With versin 0.7.0 the plugin flexform configuration has changed. To
update your configuration you have two possiblities:

#. Edit every plugin widget (booking-form, calendars, etc.).
#. Use the provided update script in the extension manager (EM).

   - TYPO3 4.5: Go to EM->Loaded Extensions->ab_booking->UPDATE!
   - TYPO3 4.6, 4.7, 6.x: go to EM->Manage Extensions->ab_booking->"Execute the Update Script" (icon)
