Typoscript Settings
-------------------

The plugin only works properly with some typoscript settings. The provided default typoscript setup does this for you but you may adjust it to your needs.

The booking form is configured by typoscript too. Please see :ref:`admin-manual-form-configuration` chapter for all settings.

.. container:: table-row

   Property
         showPrice

   Data type
         boolean

   Description
         show prices in the availability list and in the confirmation view

   Default
         1: show prices
